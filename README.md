## Introduction to Git and GitHub

### Simple Interest Calculator
A calculator that calculates simple interest given principal, annual rate of interest and time period in years.

---

### Logic & Formula
The tool follows the mathematical formula for calculating interest:

$$Simple Interest = P \times T \times R$$



### Inputs and Outputs
* **Input:**
    * `p`, principal amount
    * `t`, time period in years
    * `r`, annual rate of interest
* **Output:**
    * `simple interest = p * t * r`

---

### How to Use
1. Clone the repository: `git clone https://github.com/your-username/github-final-project.git`
2. Ensure you provide the required variables (`p`, `t`, `r`) to get the result.
